ğŸµ Adding Audio Files

Step 1: Prepare Your Audio Files

  • Convert to MP3 format (recommended for web compatibility)
  • Keep file sizes under 10MB for faster loading
  • Use descriptive, lowercase filenames with hyphens
  • Example: electronic-dreams.mp3

Step 2: Upload to assets/audio/

  • Place your MP3 files in the assets/audio/ folder
  • Make sure filenames match what's in the playlist (see script.js)
  • Current expected files:
    • electronic-dreams.mp3
    • midnight-vibes.mp3
    • urban-pulse.mp3
    • chill-waves.mp3

ğŸ–¼ï¸ Adding Cover Images

Step 1: Create Cover Images

  • Make square images (300x300px recommended)
  • Use JPG or PNG format
  • Keep file sizes under 500KB
  • Match the audio filename but with .jpg extension

Step 2: Upload to assets/images/

  • Place your cover images in the assets/images/ folder
  • Current expected files:
    • electronic-dreams.jpg
    • midnight-vibes.jpg
    • urban-pulse.jpg
    • chill-waves.jpg
