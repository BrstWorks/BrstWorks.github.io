## How to Add Your Own Audio Files:

1. **Supported Formats**: MP3, WAV, OGG, AAC
2. **Recommended**: MP3 format for best browser compatibility
3. **File Size**: Keep files under 10MB for faster loading
4. **Quality**: 128-320 kbps for good quality/size balance

## Adding New Tracks:

1. Place your audio file in this folder
2. Update the `playlist` array in `script.js` 
3. Add corresponding cover image to `../images/` folder
4. Test the player functionality
